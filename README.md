# Learn Programming by Comparison

This repository is for learning programming concepts by comparing examples written in:

- Python
- Java
- C
- C++
- Rust
- Shell Script

The main environment is AlmaLinux.

## Purpose

The goal is not only to learn individual languages, but also to understand common programming concepts by comparing how different languages express the same idea.

## Topics

1. Hello World
2. Variables
3. Conditions
4. Loops
5. Functions
6. Arrays / Lists
7. File I/O

## Environment

- AlmaLinux
- Python 3
- OpenJDK
- GCC
- G++
- Rust
- Bash
- Git / GitHub
