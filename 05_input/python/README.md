# 05_input / Python

この章では、Pythonで外部からデータを受け取る方法を学びます。

プログラムは、あらかじめ書かれた値だけで動くとは限りません。
キーボードから入力された値を使ったり、ファイルに書かれた内容を読み込んだりすることで、より実用的な処理ができるようになります。

## 学ぶ内容

- 標準入力から値を受け取る
- `input()` の基本
- 入力された文字列を数値に変換する
- 繰り返し入力する
- ファイルから文字を読み込む
- ファイルを1行ずつ読み込む

## 標準入力とは

標準入力とは、プログラムに対して外部からデータを渡す入口のことです。

多くの場合、キーボードから入力された文字が標準入力になります。

Pythonでは、`input()` を使って標準入力から文字列を受け取ることができます。

    name = input("名前を入力してください: ")
    print(f"こんにちは、{name}さん")

## input() の注意点

`input()` で受け取った値は、基本的に文字列になります。

そのため、数値として計算したい場合は、`int()` や `float()` を使って変換する必要があります。

    age_text = input("年齢を入力してください: ")
    age = int(age_text)

    print(age + 1)

## ファイルからの入力

プログラムは、キーボードからだけでなく、ファイルからデータを読み込むこともできます。

    with open("sample.txt", "r", encoding="utf-8") as file:
        text = file.read()

    print(text)

ファイルから読み込む処理は、今後さらに詳しく学びます。
この章では、「ファイルも入力の一種である」ということを確認します。

## open() のモード

`open()` の第2引数には、ファイルをどのように開くかを指定します。

たとえば、次の `"r"` は読み込み専用を意味します。

    with open("sample.txt", "r", encoding="utf-8") as file:
        text = file.read()

よく使うモードは次の通りです。

| モード | 意味 | ファイルがない場合 | 既存内容 |
|---|---|---|---|
| `"r"` | 読み込み | エラー | 残る |
| `"w"` | 書き込み | 新規作成 | 消える |
| `"a"` | 追記 | 新規作成 | 残る |
| `"x"` | 新規作成して書き込み | 新規作成 | 既にあるとエラー |

### 読み込み `"r"`

既存のファイルを読み込みます。

    with open("sample.txt", "r", encoding="utf-8") as file:
        text = file.read()

    print(text)

`"r"` は read の意味です。  
ファイルが存在しない場合はエラーになります。

### 書き込み `"w"`

ファイルに文字を書き込みます。

    with open("output.txt", "w", encoding="utf-8") as file:
        file.write("Hello\n")
        file.write("Python\n")

`"w"` は write の意味です。  
ファイルが存在しない場合は新しく作成されます。

ただし、すでに同じ名前のファイルがある場合、中身は上書きされます。

### 追記 `"a"`

既存のファイルの末尾に文字を追加します。

    with open("output.txt", "a", encoding="utf-8") as file:
        file.write("追加の行です\n")

`"a"` は append の意味です。  
すでにある内容を消さずに、最後に追加したいときに使います。

### 新規作成 `"x"`

新しいファイルを作成して書き込みます。

    with open("newfile.txt", "x", encoding="utf-8") as file:
        file.write("新しく作成しました\n")

`"x"` は新規作成専用です。  
同じ名前のファイルがすでに存在する場合はエラーになります。

間違って上書きしたくない場合に使えます。

## テキストファイルとバイナリファイル

通常の文章を扱う場合は、テキストファイルとして読み書きします。

画像や音声など、文字として扱わないファイルを読み書きする場合は、バイナリモードを使います。

    with open("image.png", "rb") as file:
        data = file.read()

    with open("copy.png", "wb") as file:
        file.write(data)

`"rb"` は binary read、`"wb"` は binary write の意味です。

この章では、まずテキストファイルの読み込みを中心に扱います。  
そのため、基本的には `"r"`、`"w"`、`"a"` を押さえておけば十分です。

## 補足：なぜ with open() を使うのか

ファイルを使うときは、本来は「開く → 読む/書く → 閉じる」という流れがあります。

この章では、安全にファイルを扱うために `with open(...) as file:` という書き方を使っています。

`with` 文を使うと、ファイルを使い終わったあとにPythonが自動的にファイルを閉じてくれます。
