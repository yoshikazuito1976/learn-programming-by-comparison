# Shell: 入力受付

この章では、Shellでキーボード入力を受け取る方法を学びます。

## 1. readで入力を受け取る

Shellでは、`read` コマンドで入力を受け取ります。

```sh
read 変数名
```

例:

```sh
echo "名前を入力してください:"
read name

echo "こんにちは、${name}さん"
```

実行例:

```text
名前を入力してください:
yoshikazu
こんにちは、yoshikazuさん
```

入力値は、変数 `name` に保存されます。

## 2. 同じ行で入力を受け取る

`echo` は表示後に改行するため、次のように書くと入力は次の行になります。

```sh
echo "名前を入力してください:"
read name
```

同じ行で入力を受け取りたい場合は、`printf` を使います。

```sh
printf "名前を入力してください: "
read name
```

実行例:

```text
名前を入力してください: yoshikazu
こんにちは、yoshikazuさん
```

`printf` は自動で改行しません。改行したい場合は `\n` を使います。

```sh
printf "こんにちは、%sさん\n" "$name"
```

## 3. Shellの変数と型

Shellでは、入力時に型を指定しません。

```sh
read age
```

たとえば `45` と入力しても、`age` に対して「整数型」「文字列型」のような型宣言はしません。

ただし、計算したい場合は `$(( ))` を使うことで数値として扱えます。

```sh
next_age=$((age + 1))
```

例:

```sh
printf "年齢を入力してください: "
read age

next_age=$((age + 1))

printf "来年は%s歳です\n" "$next_age"
```

実行例:

```text
年齢を入力してください: 45
来年は46歳です
```

## 4. 入力値には注意が必要

Shellは型を強く意識せずに書けますが、入力値が正しいとは限りません。

たとえば英字を入力した場合:

```text
年齢を入力してください: gsf
来年は1歳です
```

期待とは異なる結果になることがあります。

また、全角数字を入力すると計算でエラーになる場合があります。

```text
年齢を入力してください: ４５３
shell/input_age.sh: 行 6: ４５３: 構文エラー: オペランドが予期されます
来年は歳です
```

`４５３` は見た目は数字ですが、Shellの計算では半角数字として扱われません。

コンピュータにとって、次の2つは別の文字です。

```text
45
４５
```

## 5. この章のポイント

Shellの入力受付は次の形で書けます。

```sh
read 変数名
```

同じ行で入力を受け取りたい場合は `printf` と組み合わせます。

```sh
printf "名前を入力してください: "
read name
```

Shellでは型を明示しないため、簡単に入力を受け取れます。
一方で、計算に使う場合は、入力値が本当に数値かどうかの確認が必要です。

## 実行方法

```bash
bash hello_input.sh
bash hello_input_prompt.sh
bash input_age.sh
```

または、実行権限を付けてから実行します。

```bash
chmod +x hello_input.sh
./hello_input.sh
```
