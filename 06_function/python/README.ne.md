# 06 फङ्सन

यस अध्यायमा, प्रक्रिया (processing) लाई फङ्सनमा कसरी समेट्ने भन्ने सिकिन्छ।

उस्तै प्रकारका प्रक्रिया बारम्बार लेख्दा कार्यक्रम पढ्न गाह्रो हुन्छ।
फङ्सन प्रयोग गर्दा प्रक्रियालाई नाम दिन सकिन्छ र चाहिएको बेला बोलाउन सकिन्छ।

---

## यस अध्यायको उद्देश्य

यस अध्यायको उद्देश्य फङ्सनका आधारभूत कुरा बुझ्नु हो।

कार्यक्रम लेख्दा एउटै जस्तो प्रक्रिया पटकपटक लेख्नुपर्ने अवस्था आउँछ।

उदाहरणका लागि, तलजस्तै अभिवादन धेरै पटक देखाउनुपर्दा:

```python
print("Hello, ITO!")
print("Hello, Python!")
print("Hello, Programming!")
```

यस्तो प्रक्रिया फङ्सन प्रयोग गरेर व्यवस्थित गर्न सकिन्छ।

फङ्सनले प्रक्रियालाई नाम दिन र आवश्यक पर्दा चलाउन सजिलो बनाउँछ।

---

## कुञ्जी शब्दहरू

- फङ्सन
- परिभाषा
- कल (call)
- आर्गुमेन्ट
- फिर्ता मान
- return
- पुनःप्रयोग
- प्रक्रिया विभाजन

---

## फङ्सन भनेको के हो?

फङ्सन भनेको प्रक्रियालाई नाम दिएर समेटिएको भाग हो।

Python मा `def` प्रयोग गरेर फङ्सन परिभाषित गरिन्छ।

```python
def greet():
    print("Hello!")
```

यस उदाहरणमा `greet` नामको फङ्सन बनाइएको छ।

तर फङ्सन परिभाषित मात्र गर्दा चल्दैन।

चलाउनका लागि फङ्सनको नाम लेखेर call गर्नुपर्छ।

```python
greet()
```

---

## फङ्सनको आधारभूत रूप

Python मा फङ्सनको आधारभूत रूप यसप्रकार हुन्छ।

```python
def function_name():
    processing
```

उदाहरण:

```python
def greet():
    print("Hello!")


greet()
```

चलाउँदा नतिजा:

```text
Hello!
```

---

## फङ्सन call गर्ने

फङ्सनलाई धेरै पटक call गर्न सकिन्छ।

```python
def greet():
    print("Hello!")


greet()
greet()
greet()
```

चलाउँदा नतिजा:

```text
Hello!
Hello!
Hello!
```

उस्तै प्रक्रिया पटकपटक लेख्नुपर्दैन, फङ्सन call गरे पुग्छ।

---

## आर्गुमेन्ट भनेको के हो?

आर्गुमेन्ट भनेको फङ्सनमा पठाइने मान हो।

तलको उदाहरणमा `name` आर्गुमेन्ट हो।

```python
def greet(name):
    print(f"Hello, {name}!")


greet("ITO")
greet("Python")
```

चलाउँदा नतिजा:

```text
Hello, ITO!
Hello, Python!
```

`greet("ITO")` लेख्दा `"ITO"` मान `name` मा जान्छ।

`greet("Python")` लेख्दा `"Python"` मान `name` मा जान्छ।

---

## आर्गुमेन्ट किन प्रयोग गर्ने?

आर्गुमेन्ट प्रयोग गर्दा फङ्सनभित्र चाहिने मान बाहिरबाट पठाउन सकिन्छ।

यसले उस्तै प्रक्रियालाई सानो फरकसहित चलाउन मद्दत गर्छ।

```python
def introduce(name, language):
    print(f"{name} is learning {language}.")


introduce("ITO", "Python")
introduce("Tanaka", "Shell")
```

चलाउँदा नतिजा:

```text
ITO is learning Python.
Tanaka is learning Shell.
```

यस उदाहरणमा `name` र `language` दुई आर्गुमेन्ट प्रयोग भएका छन्।

---

## फिर्ता मान भनेको के हो?

फिर्ता मान भनेको फङ्सनले फर्काउने मान हो।

Python मा `return` प्रयोग गरेर मान फिर्ता गरिन्छ।

```python
def add(a, b):
    return a + b


result = add(3, 5)
print(result)
```

चलाउँदा नतिजा:

```text
8
```

यस उदाहरणमा `add(3, 5)` ले `8` फिर्ता गर्छ।

फर्किएको मान `result` नामको चलमा राखिएको छ।

---

## print र return बीचको फरक

`print` स्क्रिनमा देखाउन प्रयोग हुन्छ।

`return` फङ्सनको नतिजा फर्काउन प्रयोग हुन्छ।

तलको उदाहरण हेर्नुहोस्।

```python
def add_print(a, b):
    print(a + b)


def add_return(a, b):
    return a + b


add_print(3, 5)

result = add_return(3, 5)
print(result)
```

`print` ले मान देखाउँछ।

`return` ले मान फर्काउँछ।

फर्किएको मान चलमा राख्न वा अर्को गणनामा प्रयोग गर्न सकिन्छ।

---

## फिर्ता मानलाई गणनामा प्रयोग गर्ने

फिर्ता मानलाई पछि आउने प्रक्रियामा प्रयोग गर्न सकिन्छ।

```python
def add(a, b):
    return a + b


result = add(3, 5)
double_result = result * 2

print(result)
print(double_result)
```

चलाउँदा नतिजा:

```text
8
16
```

फङ्सनको नतिजा चलमा राख्दा त्यसलाई पछि सजिलै प्रयोग गर्न सकिन्छ।

---

## सर्तीय शाखा र फङ्सन

फङ्सनभित्र सर्तीय शाखा पनि प्रयोग गर्न सकिन्छ।

```python
def judge(score):
    if score >= 60:
        return "पास"
    else:
        return "फेल"


result = judge(75)
print(result)
```

चलाउँदा नतिजा:

```text
पास
```

`score` 60 वा सोभन्दा बढी भए `"पास"` फिर्ता हुन्छ।

नत्र `"फेल"` फिर्ता हुन्छ।

---

## input र फङ्सन

`input` लाई पनि फङ्सनसँग मिलाएर प्रयोग गर्न सकिन्छ।

```python
def judge(score):
    if score >= 60:
        return "पास"
    else:
        return "फेल"


score = int(input("अंक प्रविष्ट गर्नुहोस्: "))
result = judge(score)
print(result)
```

चलाउँदा उदाहरण नतिजा:

```text
अंक प्रविष्ट गर्नुहोस्: 80
पास
```

प्रविष्ट गरिएको मान प्रयोग गरेर फङ्सनले निर्णय गर्छ।

---

## दोहोर्याइ र फङ्सन

फङ्सनलाई दोहोर्याइ (loop) सँग पनि प्रयोग गर्न सकिन्छ।

```python
def judge(score):
    if score >= 60:
        return "पास"
    else:
        return "फेल"


scores = [92, 85, 74, 66, 39]

for score in scores:
    result = judge(score)
    print(f"{score}: {result}")
```

चलाउँदा नतिजा:

```text
92: पास
85: पास
74: पास
66: पास
39: फेल
```

अंकहरूको सूचीबाट एक-एक मान निकालेर फङ्सनले निर्णय गर्छ।

---

## फङ्सनमा रूपान्तरण गर्नु अघि कोड

तलको कोडले अंकको निर्णय गर्छ।

```python
score = 75

if score >= 60:
    result = "पास"
else:
    result = "फेल"

print(result)
```

यसरी पनि काम गर्छ।

तर एउटै निर्णय धेरै पटक चाहिने भए उस्तै कोड बारम्बार लेख्नुपर्छ।

---

## फङ्सनमा रूपान्तरण गरेपछि कोड

उही प्रक्रियालाई फङ्सनमा राख्दा यस्तो हुन्छ।

```python
def judge(score):
    if score >= 60:
        return "पास"
    else:
        return "फेल"


result = judge(75)
print(result)
```

फङ्सन बनाउँदा अंक निर्णय प्रक्रियालाई `judge` नाम दिन सकिन्छ।

---

## फङ्सन बनाउँदाका फाइदा

फङ्सन बनाउँदा यी फाइदा हुन्छन्।

- उस्तै प्रक्रिया पटकपटक लेख्नुपर्दैन
- कार्यक्रम पढ्न सजिलो हुन्छ
- संशोधन गर्ने ठाउँ कम हुन्छ
- प्रक्रियालाई अर्थपूर्ण नाम दिन सकिन्छ
- कार्यक्रमलाई भाग-भागमा सोच्न सकिन्छ

कार्यक्रम लामो हुँदै जाँदा फङ्सनले व्यवस्थित गर्नु झन् महत्वपूर्ण हुन्छ।

---

## फङ्सनको नाम कसरी राख्ने

फङ्सनको नामले त्यसले के गर्छ भन्ने स्पष्ट देखिनुपर्छ।

राम्रा उदाहरण:

```python
def greet(name):
    print(f"Hello, {name}!")
```

```python
def judge(score):
    if score >= 60:
        return "पास"
    else:
        return "फेल"
```

```python
def add(a, b):
    return a + b
```

त्यति राम्रो नभएका उदाहरण:

```python
def x(a):
    return a + 10
```

```python
def test(data):
    print(data)
```

अत्यन्त छोटो वा धेरै व्यापक अर्थका नामहरू पछि पढ्न गाह्रो हुन्छ।

## फङ्सन लेख्दा ध्यान दिनुपर्ने कुरा

फङ्सन लेख्दा indentation मा ध्यान दिनुहोस्।

Python मा फङ्सनभित्रका लाइनहरू indent हुनुपर्छ।

```python
def greet():
    print("Hello!")
```

तलजस्तै indent नभए error आउँछ।

```python
def greet():
print("Hello!")
```

Python मा indentation धेरै महत्वपूर्ण छ।

## अभ्यास 1: अभिवादन फङ्सन

def greet(name):

## अभ्यास 2: जोड फङ्सन

तलका सर्तहरू पूरा गर्ने फङ्सन बनाउनुहोस्।

- फङ्सन नाम: `add`
- आर्गुमेन्ट: `a` र `b`
- `a + b` को नतिजा फिर्ता गर्ने

उदाहरण:

```python
def add(a, b):
    return a + b


result = add(10, 20)
print(result)
```

## अभ्यास 3: अंक निर्णय फङ्सन

तलका सर्तहरू पूरा गर्ने फङ्सन बनाउनुहोस्।

- फङ्सन नाम: `judge`
- आर्गुमेन्ट: `score`
- 60 वा बढी भए `"पास"` फिर्ता गर्ने
- 60 भन्दा कम भए `"फेल"` फिर्ता गर्ने

उदाहरण:

```python
def judge(score):
    if score >= 60:
        return "पास"
    else:
        return "फेल"


print(judge(80))
print(judge(45))
```

## अभ्यास 4: सूची र फङ्सन

तपाईंलाई तलको अंक सूची दिइएको छ।

```python
scores = [92, 85, 74, 66, 39]
```

`judge` फङ्सन प्रयोग गरेर प्रत्येक अंकको निर्णय गर्नुहोस्।

उदाहरण:

```python
def judge(score):
    if score >= 60:
        return "पास"
    else:
        return "फेल"


scores = [92, 85, 74, 66, 39]

for score in scores:
    result = judge(score)
    print(f"{score}: {result}")
```

## अभ्यास 5: श्रेणी (Rank) निर्णय फङ्सन

अंकका आधारमा तलजस्तै श्रेणी फिर्ता गर्ने फङ्सन बनाउनुहोस्।

- 90 वा बढी: A
- 80 वा बढी: B
- 70 वा बढी: C
- 60 वा बढी: D
- 60 भन्दा कम: F

उदाहरण:

```python
def rank(score):
    if score >= 90:
        return "A"
    elif score >= 80:
        return "B"
    elif score >= 70:
        return "C"
    elif score >= 60:
        return "D"
    else:
        return "F"


scores = [92, 85, 74, 66, 39]

for score in scores:
    result = rank(score)
    print(f"{score}: {result}")
```

चलाउँदा नतिजा:

```text
92: A
85: B
74: C
66: D
39: F
```

## यस अध्यायको सारांश

यस अध्यायमा फङ्सनबारे सिकियो।

फङ्सन भनेको प्रक्रियालाई नाम दिएर समेट्ने तरिका हो।

फङ्सन प्रयोग गर्दा एउटै प्रक्रिया बारम्बार लेख्नुपर्दैन।

यसले कार्यक्रमलाई पढ्न र व्यवस्थापन गर्न सजिलो बनाउँछ।

---

## महत्वपूर्ण बुँदाहरू

- फङ्सन `def` बाट परिभाषित हुन्छ
- call नगरेसम्म फङ्सन चल्दैन
- आर्गुमेन्टले फङ्सनमा मान पठाउन मद्दत गर्छ
- `return` ले फङ्सनबाट मान फर्काउँछ
- फङ्सन प्रयोग गर्दा कार्यक्रम व्यवस्थित गर्न सजिलो हुन्छ

## अब के सिक्ने?

फङ्सन प्रयोग गर्न सिकेपछि, कार्यक्रमलाई छुट्टाछुट्टै भागको संयोजनको रूपमा सोच्न सकिन्छ।

अर्को चरणमा file input/output र error handling जस्ता विषयमार्फत लामो कार्यक्रम लेख्नेतर्फ जान्छौं।
